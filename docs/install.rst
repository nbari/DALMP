Install
=======


Clone the repository::

   $ git clone git://github.com/nbari/DALMP.git dalmp


Composer
........


A basic `composer.json <http://getcomposer.org/>`_ file:

.. code-block:: json
   :linenos:

   {
     "require": {
       "DALMP/dalmp": "*"
     },
     "minimum-stability": "dev"
   }

::
   $ composer.phar install
